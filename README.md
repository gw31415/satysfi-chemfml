# satysfi-chemfml

(English version is not available yet.)

## 概要

Easytable は  で簡素な表を組むためのパッケージです。
chemfml は [SATySFi](https://github.com/gfngfn/SATySFi) の組版で化学式や化学反応式を組みやすくするためのパッケージです。
`\chem` というコマンドを用いて、
LaTeXにおける `mhchem` のような感覚で化学式を組むことができます。
